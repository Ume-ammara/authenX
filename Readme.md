# AuthenX

**AuthenX** is a fullstack authentication system built with Node.js, PostgreSQL, and React. It provides a secure and scalable solution for user authentication and authorization in web applications.

# Features

- User registration and login
- Password hashing with bcrypt
- JWT-based authentication
- Role-based access control
- Email verification
- Password reset functionality
- OAuth 2.0 support (Google)
  -Schema validation with Zod
  -frontend built with React and Tailwind CSS
  -Frontend state with Redux Toolkit
  -Forms React Hook Form
  -Routing with Tanstack Router

# Tech Stack

- Backend
  - Node.js
    -PostgreSQL
    -Prisma ORM
    -Express.js
    -bcrypt
    -jsonwebtoken

- Frontend
  - React
  - Tailwind CSS
  - Redux Toolkit
  - React Hook Form
  - Tanstack Router
